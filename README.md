# Takealot Mid-Fidelity Wireframe Prototype

This folder contains a clickable HTML/CSS prototype for the MAF 26XS Task 2 mid-fidelity wireframe submission.

## What is included
- `index.html` - clickable prototype with 2 redesigned solutions.
- `styles.css` - design system styling and layout.
- 5 screens for Solution 1: Smart Product Discovery.
- 4 screens for Solution 2: One-Page Checkout.
- Design system section: colour palette, typography, icons/components, consistency guidelines.

## Fastest submission workflow
1. Upload this folder to a GitHub repository.
2. Go to Repository Settings > Pages.
3. Set source to the main branch and root folder.
4. Copy the GitHub Pages link and paste it into your concept note / submission.

## Figma workflow
Use one of these fast routes:
- Open `index.html` in a browser, take screenshots of each phone frame, then place them into Figma frames.
- Use the Figma plugin `html.to.design` and import the GitHub Pages URL.
- Rebuild the frames manually in Figma using the included design system and screen structure.
